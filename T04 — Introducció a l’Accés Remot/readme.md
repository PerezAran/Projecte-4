# 🌐 T04: Administració i Accés Remot

### 🛠️ Gestió de Sistemes i Infraestructura sense Barriers Físiques

---

## 🎯 Context i Objectiu de la Tasca

Com a consultors i tècnics de projectes, és indispensable trencar la barrera de la distància. Els servidors d'empresa i dels clients es troben habitualment en CPDs (Centres de Processament de Dades), armaris de comunicacions o directament al núvol.

L'objectiu d'aquesta unitat és dominar les eines d'administració remota per resoldre incidències crítiques i gestionar infraestructures de forma **segura, ràpida i eficient**, sense necessitat de desplaçaments físics.

---

## 🛠️ Eines Clau de Treball

L'estudi de l'accés remot es divideix segons el tipus d'entorn i la necessitat del servei:

### 🔒 1. SSH (Secure Shell)

* **Descripció:** El protocol estàndard per a l'administració per línia de comandes (CLI).
* **Entorn:** Utilitzat principalment en servidors Linux (com l'Ubuntu Server del projecte) i modernament també a Windows.
* **Seguretat:** Tot el trànsit, credencials i ordres viatgen de forma completament xifrada.

### 🖥️ 2. RDP (Remote Desktop Protocol)

* **Descripció:** Control total d'equips mitjançant entorn gràfic (GUI).
* **Entorn:** Eina nativa fonamental per a servidors Windows Server i sistemes d'usuari (Windows/Linux) que requereixen assistència visual.

### 🤝 3. Eines d'Assistència Ràpida (Suport a Client)

* Programari de tercers per a connexions immediates i fàbils de gestionar amb l'usuari final per al suport diari:
* 🔴 **AnyDesk**
* 💙 **TeamViewer**
* 🌐 **Chrome Remote Desktop**

## 📊 Resum de Competències a Demostrar

```
🛠️ CONNECTIVITAT  ───>  🔒 SEGURETAT  ───>  📈 EFICIÈNCIA (SLA)
Gestionar sistemes       Xifrat de dades         Resolució immediata
des de qualsevol lloc     i configuració          d'incidències

```

* **Administració Eficaç:** Gestionar qualsevol màquina com si s'estigués físicament al davant.
* **Garantia de Resposta:** Complir amb els temps de resposta pactats amb el client gràcies a la immediatesa de l'accés en línia.
* **Validació:** El domini d'aquests conceptes i protocols es comprovarà i avaluarà mitjançant una **prova teòrico-pràctica**.
